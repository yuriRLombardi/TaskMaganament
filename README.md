# TaskMaganament
Gerenciador de tarefas desenvolvido a partir de uma estrutura HTML, estilização com Bootstrap e script do Banco de Dados prévios fornecidos pelo professor da matéria "Programação Web II", para aplicar os conhecimentos em PHP, Orientação a Objetos e PDO. Projeto desenvolvido no 2º ano do Técnico de Informática Integrado ao Ensino Médio.
